Biopython is an open source set of tools for biological computation written in Python by an international team of developers.

Different modules extend Biopython's capabilities to include many biological features such as:

  • Sequence Annotation          Genome Diagrams          Machine Learning
  • Phylogeny          Population Genetics          Protein Structure

Summary

Over the course of 140 hours I've been able to contribute new functions to several different biopython modules in addition to general bug fixes, documentation, and writing tests. Through this I've gained experience in how to communicate with scientists and developers to work together and produce maintainable code.

Following a strict format was new to me as I've only ever used git for personal project or group course work where we had more freedom to do as we will. Running tests to ensure the code was compatible with all supported python versions takes nearly 2 hours, this taught me to be more careful before making a pull request or updating any code as to not waste wait time for testing or block others in the queue.

All of the contributions I've been able to make were thoroughly checked by hand by the maintainers or another collaborator in addition to automated testing. At the start I expected that having to wait for feedback would be one of the more time consuming parts of this project, but the maintainer and most members are very active in the community and very helpful with giving feedback.

I've learned how to write concise, clean, and well commented code. There was also an emphasis on efficiency whenever possible.

This was the first instance where I've had to write so many unittests. This helped a lot with debugging and having to think about the many different cases of failure. I know that these tests will also help in the future to have a basis on what functionality the method must maintain if it ever comes time to update it.

Since I had originally started on this project last term in September of the Fall 2018 term before switching it to the Spring 2019 term, I was able to have some of my contributions be a part of the Biopython 1.73 release.
