Biopython

Biopython is an open source set of tools for biological computation written in Python by an international team of developers.

Different modules extend Biopython's capabilities to include many biological features :

  • Sequence Annotation
  • Phylogeny
  • Genome Diagrams
  • Population Genetics

  • .
  • Machine Learning
  • Protein Structure
